# Dental Website - Complete Solution

## Features
- ✅ No PHP Required (Pure HTML/CSS/JS)
- ✅ GitHub Pages Compatible
- ✅ Formspree Integration
- ✅ LocalStorage Based Admin Panel
- ✅ Fully Responsive
- ✅ SEO Optimized
- ✅ Base64 Image Storage

## Quick Start

### 1. GitHub Pages Deployment
1. Create new repository on GitHub
2. Upload all files
3. Go to Settings → Pages
4. Select "Deploy from Branch" → "main" → "/ (root)"
5. Your site will be live at `https://yourusername.github.io/repository-name/`

### 2. Admin Panel Access
- Go to `your-site.com/admin.html`
- Default password: `admin123`
- **IMPORTANT**: Change password in `admin.js` before deploying

### 3. Formspree Setup
1. Create account at [formspree.io](https://formspree.io)
2. Create new form
3. Copy Form ID
4. Paste in Admin Panel → Form Settings

### 4. Customization
1. Login to Admin Panel
2. Update all settings:
   - General info (clinic name, doctor, contact)
   - Colors (primary, secondary, accent)
   - Services (add/edit/delete)
   - Reviews (patient testimonials)
   - Social media links
   - SEO meta tags
   - Images (logo, favicon, hero, doctor photo)

### 5. Data Management
- **Export**: Admin Panel → Dashboard → Export Data (creates JSON backup)
- **Import**: Admin Panel → Dashboard → Import Data (restore from JSON)
- **Reset**: Admin Panel → Dashboard → Reset All (factory reset)
